# Contributing

Thank you for your interest in contributing to Copilot-by-Arch.

This project is currently focused on documenting AI-assisted development workflows on Arch Linux.

## Ways to Contribute

You can contribute by:

- Improving setup instructions
- Adding configuration examples
- Fixing documentation errors
- Adding troubleshooting notes
- Reporting issues
- Suggesting improvements

## Contribution Guidelines

Before submitting a pull request, please make sure that:

- No API keys, tokens, passwords, or private information are included
- Documentation is clear and reproducible
- Commands have been tested where possible
- Changes are relevant to Arch Linux or AI-assisted development workflows

## Pull Requests

Please describe:

- What was changed
- Why the change is useful
- How it was tested, if applicable

## Issues

When opening an issue, please include:

- Your operating system
- Relevant package versions
- Steps to reproduce the problem
- Expected behavior
- Actual behavior
