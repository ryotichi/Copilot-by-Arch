# Example Configuration

This file shows an example structure for documenting local AI-assisted development settings.

Do not include real API keys, access tokens, passwords, or private information.

## Example

```text
EDITOR=your-editor
PROJECT_DIR=~/projects
AI_ASSISTANT_ENABLED=true
LOG_LEVEL=info
```

## Notes

Use this file only as a template.
Create your own local configuration file outside the repository when storing private values.
